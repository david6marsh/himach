# Antarctic-centred coordinate reference system

Coordinate reference system (CRS) for plotting and analysing maps. WGS
84 / Antarctic Polar Stereographic. Used in analysis, but not
recommended for plots.

## Usage

``` r
crs_S
```

## Format

CRS

## Details

crs_N is EPSG 3031
