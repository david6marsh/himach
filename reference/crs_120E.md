# Asia-centred coordinate reference system

Coordinate reference system (CRS) for plotting and analysing maps.
Centred on East Asia (120E).

## Usage

``` r
crs_120E
```

## Format

CRS

## Details

"+proj=robin +lon_0=120 +x_0=0 +y_0=0 +ellps=WGS84 +datum=WGS84 +units=m
+no_defs"
