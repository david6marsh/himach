# Atlantic-centred coordinate reference system

Coordinate reference system (CRS) for plotting and analysing maps.
Atlantic-centred. Works for most analysis, but not recommended for
N-region (eg New Zealand and Fiji), instead use
[`crs_Pacific`](https://david6marsh.github.io/himach/reference/crs_Pacific.md).

## Usage

``` r
crs_Atlantic
```

## Format

CRS

## Details

crs_Atlantic is "+proj=robin +lon_0=0 +x_0=0 +y_0=0 +ellps=WGS84
+datum=WGS84 +units=m +no_defs"
