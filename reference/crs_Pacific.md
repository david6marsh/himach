# Pacific-centred coordinate reference system

Coordinate reference system (CRS) for plotting and analysing maps.
Pacific-centred.

## Usage

``` r
crs_Pacific
```

## Format

CRS

## Details

"+proj=robin +lon_0=180 +x_0=0 +y_0=0 +ellps=WGS84 +datum=WGS84 +units=m
+no_defs"
