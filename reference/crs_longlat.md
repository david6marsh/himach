# Lat-long coordinate reference system

Coordinate reference system (CRS) for creating maps from
longitude-latitude coordinates. Used in analysis, but not recommended
for plots.

## Usage

``` r
crs_longlat
```

## Format

CRS

## Details

`crs_longlat` is EPSG4326
